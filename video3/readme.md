# Video 3: Conversational Agents

This folder contains all the code from Video 3 of the "AI in Testing" series.

## 📁 Files Overview

1. **counter_no_state.py**  - Streamlit with no Memory
2. **counter_with_state.py** - Streamlit with Memory
3. **chat_testing_agent.py** - Complete Conversational Test Agent


## 🚀 Setup Instructions

### 1. Prerequisites
- Python 3.10 or higher installed
- Ollama installed and running (from Video 1)
- A code editor (VS Code or PyCharm recommended)

### 2. Install Dependencies

```powershell
pip install -r requirements.txt
```


### 3. Verify Ollama is Running

Open browser and go to: http://localhost:11434

You should see: "Ollama is running"

## 4. Running the Scripts

streamlit run counter_no_state.py
streamlit run counter_with_state.py
streamlit run chat_testing_agent.py


```
Interactive web interface for your LLM testing assistant. Opens automatically in your browser at http://localhost:8501
